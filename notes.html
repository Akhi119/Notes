HTML

To perform html operations and create web pages we have plethora of IDE like atom,visual code, notepad++,notepad,Intellij etc.. Basically, HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. It describes the structure of a Web page It consists of a series of elements

The Html elements tell the browser how to display the content.
  • Tag html tells element is the root element of an HTML page tells to the Browser that everything between the opening, and closing tag will be an HTML.
  • The tag head element contains meta information about the HTML page.
  • The tag title element displays the title of the page
  • The body element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The tag h1 element defines a large heading
  • The tag p element defines a paragraph
  • The tag b elemet is for break
  • The tag centre is to keep the entire text in centre

    • All HTML elements can have attributes Attributes provide additional information about elements Attributes are always specified in the start tag Attributes usually come in name/value pairs like: name="value"

    CSS

    CSS is Cascading Style Sheet.It is used for defining the styles for web pages. It describes the look and formatting of a document which is written in a markup language. It provides an additional feature to HTML. It is generally used with HTML to change the style of web pages and user interfaces.

    Uses Of CSS

  • Solve problems easily
  • Saves lot of time
  • provide more attributes
  • loads faster

    • Some of the CSS property

  • CSS Border-The Border style property is used to specify the border type which you want to display on the web page.
  • background color-The background-color property is used to specify the background color of the element.
  • margin-The margin property sets the margins for an element
  • Text-Align -The text-align property specifies the horizontal alignment of text in an element.

    • Types of CSS:

    Inline CSS

  • An inline style may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

    • Internal CSS

  • An internal CSS is used to define a style for a single HTML page An internal CSS is defined in the head section of an HTML page, within a style element

    • Style Tag

    Example
    <style type=”text/css”> </style>

    External CSS

  • An external style sheet is used to define the style for many HTML pages. To use an external style sheet, add a link to it in the head section of each HTML page. The external style sheet can be written in any text editor. The file must not contain any HTML code, and must be saved with a .css extension.

    • Uses the Link tag

    Example
      <link rel=”stylesheet” href=”../morin.css”
 type=”text/css”>

    • Css Syntax

    A CSS comprises of style rules that are interpreted by the browser and then applied to the corresponding elements in your document

    A style rule is made of three parts:
  • Selector:A selector is an HTML tag at which a style will be applied. This could be any tag like h1 or table etc.
  • Property:: A property is a type of attribute of HTML tag. Put simply, all the HTML attributes are converted into CSS properties. They could be color, border, etc
  • Value:: Values are assigned to properties. For example, color property can have the value either red or #F1F1F1 etc.
    • You can put CSS Style Rule Syntax as follows

    selector { property: value }

    Some of the Font Properties are below

    Type Face- Font Familly

  • Consists of a comma-separated list of font names
    • Example

 “ Luxi Serif”,
 “ DejaVu LGC Serif”,

    Type Face- Font Size

    Specified in one of the CSS size units

    1. 1em is the width of a letter m (relative to surrounding text)
    2. 1pt is a standard typographic point (1/72 inches)
    3. 1px is one screen pixel
    4. Keywords: xx-small, x-small, small, medium, large, x-large, xx-large are browser dependant
    5. Percentages: XXX% relative to the surrounding code
    6. Points and pixels are extremely accurate but not resizeable
    Example
    h1 { font-size: 200%; }
h2 { font-size: large; }
h3 { font-size: medium; }

    Text Properties

    Some of the Text Properties are below

    Text properties - Color

  • Can be a color name, a rgb value, or a hexadecimal value
  • You can find multiple color names at Colorhunt

    • Example
    /* Make all headings pure red */
H1 { color: red; }
H2 { color: rgb(255,0,0); }
H3 { color: #ff0000; }

    Text properties - Background

  • The background property specifies the background color and/or image
  • background-color specifies a color (named, RGB, or hex)
  • background-image, background-position, background-repeat, and/or backgroundattachment to specify a background image
  • Usually used on the body tag

    • Text properties - Text Align

  • Determine which side of the page (or containing box) that text lines up with
  • Can be one of left, right, center, or justify
    • Example
    h1 { text-align: center; }
p { text-align: left; }

    Text properties - Text Decoration

  • Add some decoration to the text
  • Can be one of none, underline, overline, line-through, or blink
    • Example
    h1 { text-decoration: underline; }
a { text-decoration: none; }

    Text properties - Text Transform

  • Can actually transform the text
    • Example
    h1 { font-size: large;
 text-transform: capitalize;
}
h2 { font-size: normal;
 text-transform: uppercase;
}
